# Day 2 Exercises

This directory contains hands-on exercises and example specs for Day 2 of the workshop.

## Files

- `exercises.md` - List of exercises for participants to complete during Day 2 hands-on sessions

## Purpose

These exercises help participants practice:
- Reviewing and improving AI-generated tests
- Working with Playwright agents (Planner, Generator, Healer)
- CI/CD integration and artifact management
- Extending GitHub Actions workflows

The exercises are organized by parts:
- Part 1: AI-powered edge cases
- Part 2: Playwright agents
- Part 3: CI/CD
